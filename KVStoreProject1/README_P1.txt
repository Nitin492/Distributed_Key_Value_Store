Description: 
kvclient.java accepts subcommands and identifies them as get, set and del by splitting the incoming args.
kvserver.java starts the server.
KVStoreHandler.java handles the commands and gives appropriate Results. It mainntains a hashmap to store the data.

OS: Windows 
IDE: Eclipse
Languages: Java 
Libraries:  libthrift-0.9.3.jar, slf4j-api-1.7.12.jar, slf4j-simple-1.7.12.jar

Images are located in the 'Screenshots' folder

List of source code files in 'src/kvstore' folder:
kvclient.java
kvserver.java
KVStoreHandler.java 
Result.java (auto-generated by thrift)
KVStore.java (auto-generated by thrift)
ErrorCode.java (auto-generated by thrift)

Running commands:
Go to the folder 'Executables' where kvserver.jar and kvclient.jar are located.

java -jar kvserver.jar (Starts the Server)
java -jar kvclient.jar -server localhost:9090 -set x 20 (Sets key-value pair)
java -jar kvclient.jar -server localhost:9090 -get x (Gets the value for specified key)
java -jar kvclient.jar -server localhost:9090 -del x (Deletes the specified key-value pair)


Note: Sequence of arguments is important.

Using the Code:(Use Eclipse)
Create empty Java project in Eclipse.
Right click the empty project and select import
In pop-up box: General -> FileSystem
Navigate to location of KVStore folder and Import it.

For Compilation: 
File -> Export Select runnable JAR file.
launch configuration = kvserver - KVStore (for server - kvserver.jar)
launch configuration = kvclient - KVStore (for client - kvclient.jar)
Specify the location and name as kvserver.jar and kvclient.jar respectively.
Click Finish.
